# Contributing

Thanks for your interest in CodexLens. Issues and pull requests in Chinese or English are both welcome.

## Before You Start

- For non-trivial changes, please open an issue first so we can confirm direction and scope.
- Keep changes focused. Avoid unrelated refactors in the same pull request.
- Do not include secrets, local database files, or unredacted session content in issues, screenshots, logs, or commits.

## Local Development

```bash
npm install
npm run dev
npm run build
npm run test
```

## Pull Requests

- Explain what changed and why.
- List the validation you actually ran.
- Add screenshots for UI changes when helpful.
- Update documentation when behavior, UX, or commands change.

## Privacy Notes

CodexLens works with local Codex session data. Please redact usernames, local paths, prompts, and any sensitive conversation content before sharing anything publicly.
